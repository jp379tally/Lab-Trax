# SEO Strategy

## In scope
- Public root landing page served by `artifacts/labtrax/server/serve.js` at `/`
- Public desktop web shell served at `/desktop/`
- Crawlability files and crawler governance for the deployment root (`robots.txt`, `sitemap.xml`, related bot guidance)

## Out of scope
- Authenticated desktop SPA routes after sign-in
- Native mobile app screens and app-store listing metadata
- JSON API endpoints and binary download/file routes except where they affect crawler discovery

## Target audience
- Dental labs, dental practices, and providers evaluating or accessing LabTrax.

## Primary keywords
- LabTrax
- dental lab management
- dental lab case tracking
- dental lab operations
